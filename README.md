# portal-migusport-pages

## 项目简介

这是一个用于归档和发布独立 HTML 页面的仓库。仓库内包含多个不依赖于特定域名的静态页面，适用于页面存档、临时发布或内容展示等场景。所有页面均为独立文件，无需后端支持，可直接通过浏览器打开或部署到静态托管服务。

## 目录说明

```
.
├── pages/           # 存放所有 HTML 页面文件
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/          # 公共资源文件（如 CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
├── index.html       # 可选的主索引页面
└── README.md        # 本文件
```

- `pages/`：所有独立 HTML 页面存放于此，每个页面为一个完整的功能单元。
- `assets/`：存放页面间可能共用的静态资源，便于统一管理和引用。
- `index.html`：可选的入口页面，用于快速导航或展示页面列表。

## 页面归档说明

- 每个页面均为自包含的 HTML 文件，内部引用资源路径建议使用相对路径。
- 页面内容不绑定任何特定域名或服务，可自由迁移至任何静态托管环境。
- 归档的页面可能包含测试、演示或临时内容，请以实际文件为准。
- 页面更新或删除时，请同步更新本 README 及索引页面（如有）。

## 维护说明

- 新增页面：将 HTML 文件放入 `pages/` 目录，如有公共资源请放入 `assets/` 对应子目录。
- 删除页面：从 `pages/` 移除文件，并检查是否有孤立资源需要清理。
- 修改页面：直接编辑对应 HTML 文件，注意保持资源路径正确。
- 提交前请检查页面是否可正常打开，避免引用丢失或路径错误。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器打开 `pages/` 下的任一 HTML 文件即可查看。
3. 如需部署，可将整个仓库上传至任意静态文件服务器或托管平台。

## 许可证

本仓库内容按需使用，未明确声明许可证的页面默认仅用于归档展示。如有特殊版权要求，请参考页面内注释或联系维护者。
